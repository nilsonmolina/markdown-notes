# Disk Usage
Get file size of a folder/file
```
$ du -hs <file/folder>
```
List all folders, their size and give a total: 
```
$ du -hsc *
```

# Shutdown 
To shutdown now
```
$ sudo shutdown -h now
```
To restart now
```
$ sudo shutdown -r now
```
To shutdown in 60 minutes
```
$ sudo shutdown -h +60
```
To restart in 5 minutes
```
$ sudo shutdown -r +5
```
Cancel scheduled shutdown
```
$ shutdown -c
```

# Show Date & Time
To show the current date and time
```
$ date
```
